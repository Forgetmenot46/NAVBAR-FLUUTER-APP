# JAIFULFOOD k6

ยินดีต้อนรับสู่แอพ JAIFULFOOD k6! แอพนี้ถูกพัฒนาขึ้นเพื่อแนะนำร้านอาหารในระแวกคลอง 6

## ฟีเจอร์

- แนะนำร้านอาหารในคลอง 6
- แสดงรายละเอียดของแต่ละร้าน
- ค้นหาร้านอาหารตามประเภท
- รีวิวและให้คะแนนร้านอาหาร
